# 开发说明

## 1. 后端分层建议

- `controller`：接口路由与参数校验
- `service`：业务逻辑
- `repository`：数据库访问（后续接入 MyBatis/JPA）
- `config`：跨域、安全、序列化等配置
- `sql`：数据库初始化脚本

## 2. RAG 接入建议

- 当前实现：基于文档关键词重叠的轻量检索 + 可解释引用
- 可升级路线：
  - 文档上传 -> 切块 -> 向量化 -> 入库
  - 向量召回 + 关键词召回 + 重排
  - Prompt 模板 + 引用约束 + 置信度评分

## 3. 数据层扩展建议

- 接入 MySQL 8 存储业务数据
- 接入 Redis 缓存热点查询
- 接入向量库（Milvus 或 FAISS）
- 当前提供 `backend/sql/schema.sql` 与 `backend/sql/seed.sql` 用于初始化

## 4. 测试建议

- 后端：JUnit + MockMvc 做接口测试
- 前端：Vitest + Vue Test Utils 做组件测试
- 性能：JMeter 压测关键接口
