# 接口清单（骨架版）

## 0. 登录接口

- 方法：`POST`
- 路径：`/api/auth/login`
- 请求体：

```json
{
  "username": "admin",
  "password": "admin123"
}
```

- 返回体：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "token": "jwt-token",
    "username": "admin",
    "role": "ADMIN"
  }
}
```

- 说明：除 `/api/health` 与 `/api/auth/login` 外，其余接口需在请求头带 `X-Auth-Token`，其值为登录返回的 JWT。

## 1. 健康检查

- 方法：`GET`
- 路径：`/api/health`
- 描述：后端可用性检测

## 2. 问答接口

- 方法：`POST`
- 路径：`/api/rag/ask`
- 请求体：

```json
{
  "question": "客户投诉房间异味如何处理？"
}
```

- 返回体：

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "answer": "示例回答",
    "citations": ["文档A", "文档B"]
  }
}
```

## 3. 运营看板

- 方法：`GET`
- 路径：`/api/ops/dashboard`
- 描述：返回入住率、营收、评分、差评率及建议文本

## 4. 知识库管理

- `POST /api/kb/docs`：新增文档
- `GET /api/kb/docs`：查询全部文档
- `GET /api/kb/search?q=关键词`：按关键词检索文档
