# 接口测试用例

## 1. 登录接口

- 接口：`POST /api/auth/login`
- 用例 1：正确账号密码
  - 输入：`admin/admin123`
  - 预期：`code=0`，返回 JWT
- 用例 2：错误密码
  - 输入：`admin/xxx`
  - 预期：`code=401`

## 2. 问答接口

- 接口：`POST /api/rag/ask`
- 前置条件：已登录，Header 带 `X-Auth-Token`
- 用例 1：常见投诉问题
  - 输入：`客户投诉异味怎么处理`
  - 预期：返回答案与 citations 数组
- 用例 2：无认证信息
  - 预期：`code=401`

## 3. 知识库接口

- 接口：`POST /api/kb/docs`
- 用例 1：新增正常文档
  - 输入：标题+内容
  - 预期：新增成功，返回文档 ID
- 用例 2：内容为空
  - 预期：`code=400`

- 接口：`GET /api/kb/docs`
- 用例：获取文档列表
  - 预期：返回数组且长度 >= 初始化文档数量

## 4. 看板接口

- 接口：`GET /api/ops/dashboard`
- 用例：已登录访问
  - 预期：返回 occupancyRate、dailyRevenue、trend 字段
