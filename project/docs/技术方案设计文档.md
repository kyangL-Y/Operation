# 技术方案设计文档

## 1. 总体架构

- 前端：Vue3 + Vite
- 后端：SpringBoot 3.x
- 数据层：当前为内存 + SQL 脚本，后续接入 MySQL/向量库
- 检索层：当前关键词检索，后续升级向量检索 + 重排

## 2. 后端模块

- `controller`：`AuthController`、`RagController`、`KnowledgeController`、`OpsController`
- `service`：认证、知识检索、问答生成
- `config`：跨域、鉴权拦截、全局异常处理
- `common`：统一响应体与业务异常

## 3. 前端模块

- `LoginView`：登录获取 JWT
- `AskView`：问答请求与引用展示
- `KnowledgeView`：知识新增与列表查询
- `DashboardView`：运营指标与趋势展示

## 4. 鉴权设计

- 登录接口返回 JWT
- 前端请求拦截器自动附加 `X-Auth-Token`
- 后端拦截器校验 JWT，未通过返回 401

## 5. RAG 设计（当前与升级）

当前实现：

- 文档入库（内存）
- 关键词匹配打分
- 输出命中片段与引用 ID

升级路线：

- 文档切块与向量化
- 混合召回（BM25 + 向量）
- 重排与生成约束
- 引用可信度评分

## 6. 数据模型（SQL 草案）

- `sys_user`
- `kb_document`
- `ops_daily_metric`

详见：`backend/sql/schema.sql`
